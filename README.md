# PPTRolePlaying
基于开源公司框架的游戏公司项目，目标是推动游戏行业的创造力生产工具的平民化。  
当前的具体项目是一个剧情演绎的游戏项目，玩法像短视频一样，点击窗口就可加入匹配进行剧情，每个人都可以像发布视频一样发布剧情。
